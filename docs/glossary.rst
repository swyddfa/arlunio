.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   LHS
      The left hand side of an equation/inequality/etc. E.g. in the equation
      :math:`a^2 + b^2 = c^2`, the left hand side refers to the expression
      :math:`a^2 + b^2`

   sdist
      Shorthand for a python source distribution

   tox
      `tox <https://tox.readthedocs.io/en/latest/>`_ is an automation tool for the setup
      of multiple python envrionments and running commands within them. Very often used
      for testing.

   wheel
      A `wheel <https://pythonwheels.com/>`_ is the preferred packaging format for
      Python code
