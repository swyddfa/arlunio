User GuideÂ¶

Welcome to the user guide! It is made up from a number of guided projects that each try to focus on a particular aspect of arlunio. Each project should be self contained so feel free to try them in whichever order you fancy.

Note

There is also an interactive version of the user guide available if you have arlunio setup on your machine. It can be launched by running the command:

$ arlunio tutorial

This will open a Jupyter Lab instance pointed at this section of the documentation.
