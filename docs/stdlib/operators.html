Operators¶

Built in operators.

Note

It’s often not necessary to use these definitions directly since they plug into the operator system.

MaskAdd¶

class arlunio.lib.operators.MaskAdd(*, a: Defn[Mask] = None, b: Defn[Mask] = None)[source]¶

Derives from: width, height

Add any two mask producing definitions together.

The resulting defintion will return a mask that is True if a given point is True in either a or b.

a¶

The first mask

b¶

The second mask

Examples

This definition is used implicitly when adding two masks together

import arlunio as ar
from arlunio.lib import Circle, Square

c1 = Square(xc=-0.25, yc=-0.25, size=0.55)
c2 = Circle(xc=0.25, yc=0.25, r=0.7)

c = c1 + c2
image = ar.fill(c(1920,1080))

Or can be used directly

import arlunio as ar
from arlunio.lib import Circle, MaskAdd, Square

a = Square(xc=-0.25, yc=-0.25, size=0.55)
b = Circle(xc=0.25, yc=0.25, r=0.7)

c = MaskAdd(a=a, b=b)
image = ar.fill(c(1920, 1080))

MaskMul¶

class arlunio.lib.operators.MaskMul(*, a: Defn[Mask] = None, b: Defn[Mask] = None)[source]¶

Derives from: width, height

Muliply any two mask producing definitions together.

The resulting definition will return a mask that is True only if a given point is in both a and b.

a¶

The first mask

b¶

The second mask

Examples

This definition is used implicitly when multiplying two masks together

import arlunio as ar
from arlunio.lib import Circle, Square

c1 = Square(xc=-0.25, yc=-0.25, size=0.55)
c2 = Circle(xc=0.25, yc=0.25, r=0.7)

c = c1 * c2
image = ar.fill(c(1920,1080))

Or can be used directly

import arlunio as ar
from arlunio.lib import Circle, MaskMul, Square

a = Square(xc=-0.25, yc=-0.25, size=0.55)
b = Circle(xc=0.25, yc=0.25, r=0.7)

c = MaskMul(a=a, b=b)
image = ar.fill(c(1920, 1080))

MaskSub¶

class arlunio.lib.operators.MaskSub(*, a: Defn[Mask] = None, b: Defn[Mask] = None)[source]¶

Derives from: width, height

Subtract one mask away from another mask.

The resulting definition will return a mask that is True only if a given point is in a and not in b.

Note

You will get very different results depending on which way around you set a and b!

a¶

The first “base” mask

b¶

The second mask that defines the region to remove from a

Examples

This definition is used implicitly when subtracting one mask from another

import arlunio as ar
from arlunio.lib import Circle, Square

c1 = Square(xc=-0.25, yc=-0.25, size=0.55)
c2 = Circle(xc=0.25, yc=0.25, r=0.7)

c = c1 - c2
image = ar.fill(c(1920,1080))

Or can be used directly

import arlunio as ar
from arlunio.lib import Circle, MaskMul, Square

a = Square(xc=-0.25, yc=-0.25, size=0.55)
b = Circle(xc=0.25, yc=0.25, r=0.7)

c = MaskMul(a=b, b=a)
image = ar.fill(c(1920, 1080))
